#### 一、标题
#:个数代表了几级，最高6级
####  二、字体
1. 加粗：文字左右用**包起来
2. 斜体：要倾斜的文字左右用一个*包起来
3. 斜体加粗：文字左右用三个*包起来
4. 删除线：文字左右用两个~包起来
```
**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```

#### 五、图片
```
![图片alt](图片地址)
图片alt就是图片不能正常显示时，对图片的一个解释
```
#### 六、超链接
```
[超链接名](超链接地址)
```
#### 七、列表
1. 无序列表

```
- 列表内容
+ 列表内容
* 列表内容
```
2. 有序列表

```
1.. 列表内容
3. 列表内容
4. 列表内容
数字和内容有空格
```
3. 列表嵌套

```
+ 一级列表
   - 2
   - 2
   - 2
+ 一级列表
      1. 列表
      2. 列表
      3. 列表
+ 一级列表
```
