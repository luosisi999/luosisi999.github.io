git入门使用

发表于 2016-11-01   |   分类于 git基本使用
bash的常见命令
  1. pwd 查看当前的目录
  2. cd 切换目录
  3. ls 查看当前目录下的内容
    • ls -a 查看当前目录下的所有文件
    • ls -al 查看当前目录下的所有文件的详细信息
  4. mkdir 创建一个目录
  5. touch 创建一个文件
  6. cat 查看文件的内容
  7. less/more 查看当前文件的内容（分页显示）
  8. rm “文件名” 删除当前文件(空文件，不常用)
  9. rm -r “文件名” 递归的将整个文件删除
  10. rm -rl “文件名” 强制的将整个文件都删除
  11. mv “文件名” “目的地” （可以改名）
  12. cp “文件名” “目的地” 复制文件 （可以改名）
  13. head 查看文件前面内容 eg:head -10 index.html
  14. tail 查看文件后面内容 eg:tail -10 index.html
  15. tab 自动补全
  16. history 查看历史命令
  17. whoami 查看当前用户
  18. wget 下载
  19. ssh 远程登录 ssh 账户@ip地址
  20. “>” 或”>>” 重定向
  21. grep 匹配内容，一般结合管道符使用
  22. | 管道符可以将多个命令连接使用，上一次（命令）的执行结果当成下一次（命令）的参数
vi编辑器

1.三种模式

+ 命令行模式 
+ 输入模式    
+ 末行模式

2、使用vi编辑器
a) 打开/创建文件， vi 文件路径
b) 底行模式 :w保存，:w filenme另存为
c) 底行模式 :q退出
d) 底行模式 :wq保存并退出
e) 底行模式 :e! 撤销更改，返回到上一次保存的状态
f) 底行模式 :q! 不保存强制退出
g) 底行模式 :set nu 设置行号
h) 命令模式 ZZ（大写）保存并退出
i) 命令模式 u辙销操作，可多次使用
j) 命令模式 dd删除当前行
k) 命令模式 yy复制当前行
l) 命令模式 p 粘贴内容
m) 命令模式 ctrl+f向前翻页
n) 命令模式 ctrl+b向后翻页
o) 命令模式 i进入编辑模式，当前光标处插入
p) 命令模式 a进入编辑模式，当前光标后插入
q) 命令模式 A进入编辑模式，光标移动到行尾
r) 命令模式 o进入编辑模式，当前行下面插入新行
s) 命令模式 O进入编辑模式，当前行上面插入新行

git常用命令
  1. git config –global user.name “名字” ;
  2. gir config –global us
  3. git init 格式化该目录文件为git文件
  4. git add “文件名” 将修改的文件添加到缓存区
    • git add –all
    • git add -A
    • git add *
  5. git commit -m “提交说明”
  6. git log 查看提交日志
  7. git status 查看当前文件的状态
  8. git checkout “文件名” 将文件从缓存区忘外剪除
  9. git reset –hard “sha值”
  10. git branch 查看分支
  11. git branch “分支名” 添加分支
  12. git checkout “分支名” 切换分支名
  13. git merge 分支名 合并分支
  14. ssh-keygen -t rsa 创建公钥，私钥（电脑用户的.ssh下）
  15. git remote 命名 .git仓库的路径 (添加一个本地的远程仓库，并命名为rsa)
  16. git branch -d 分支名 删除本地的分支
  17. git branck -a 查看所有的分支，包括远程分支
  18. git remote 查看添加的多个远程的主机名
  19. git remote show 主机名 查看某个指定的主机名
  20. git push 主机名/地址 分支
  21. git pill 主机名/地址 分支
git较高级命令
  1. 忽略文件
    • touch .gitignore
    • 说明：该文件用来配置那些文件不用上传到远程仓库
    • eg:.jpg,.txt 或者是具体的文件名
  2. 比较差异
    • git difftool 比较缓存区和工作目录的区别
    • git difftool sha值 比较工作目录和该sha值版本的区别
    • git difftool sha值 sha值 比较两个sha值版本的区别
    • git difftool 分支 比较工作目录和该分支的区别
