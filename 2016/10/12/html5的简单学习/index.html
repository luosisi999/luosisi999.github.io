html5的简单学习

html5的概念

我们日常讨论的的 h5其实是一个泛称，它指的是h5+css3+javascript
等技术组合成的应`用开发平台`。

h5的优点

较之前的版本更简洁，更宽松。

新增的标签(常用的语义标签）

  • nav 导航
  • header 页眉
  • footer 页脚
  • section 区块
  • article 文章
  • aside 侧边栏
  • progress进度条

兼容性处理

在不兼容的浏览器里面可以通过document.creatElement("tagName")
来处理兼容。
但是我们在实际开发中一般使用第三方提供的js库来解决兼容，原理同上者一样。

新增的表单

  • email 输入邮箱的格式
  • tel 输入手机的格式
  • number 输入数字格式
  • url 输入url的格式
  • range 自由拖动滑块
  • color 拾色器
  • time 时间 时：分
  • date 日期 年/月/日
  • datetime 日期和时间 但是浏览器不支持
  • month 月份 年–月
  • week 星期 年–周

eg:<input type=”email”> 其他的也是一样的格式

新增的表单元素

  • <datalist> 数据列表
  • <keygen > 生成加密字符串
  • <output > 输出结果
  • <meter > 度量器

表单的属性

  • placeholder 占位符
  • autofocus 自动获得焦点
  • mutiple 多文件上传
  • autocomplete 自动完成提示
  • form 将该表单标签自动提交到指定标签
  • novalidate 关闭验证
  • required 必须填
  • pattern 自定义验证（通过正则表达式）

音频

<audio src=”1.mp3 controls”>

属性：autoplay 自动播放

controls 是否显示自动播放控件 这是一个必写属性

loop 表示循环播放 取值1,2,3…loop=loop表示无限循环

注意，音频的兼容问题

解决方案：

一般我们是通过source标签来引入不同格式的音频来处理兼容问题

<audio controls >

< source src=”1.mp3”>

< source src=”1.wav”>

< source src=”1.ogg”>

你的浏览器不支持html音频播放功能

</audio>

视频

<vidio src=”1.mp4” controls><video >

属性

autoplay 自动播放
loop 循环播放
width 设置窗口的宽度
height 设置窗口的高度

兼容

<video> controls=”controls” >
< source src=”1.mp4”>

< source src=”1.wav”>

< source src=”1.ogg”>

你的浏览器不支持html视频播放功能
</video>

DOM的扩展

1.获取元素

1.0 document.getElementsByClassName(“class”) 通过类名来获取元素，以伪数组的形式存在。

2.0 document.querySelector(“selector”)通过css选择器来获取元素，返回符合条件的第一个元素。

3.0 documentquerySelectorAll(“selector”) 通过css 选择器来获取元素，是以一个伪数组的形式存在。

2.类名操作

+　node.classList.add(“”class”) 添加class

  • node.classList.remove(“class”) 移除class
  • node.classList.toggle(“class”) 切换class.有则移除，无则添加
  • node.classList.contains(“”) 检测是否存在class

自定义属性(data-=)

在h5里面我们可以自定义的添加属性，格式data-*=””,列如，data-info=”我是自定义的属性”，通过Node。dataset[info],这样我们就可以获取到自定义的值。

node.dataset是以对象的形式存在的，当我们为一个dom节点添加了多个自定义属性时。node.dataset则存储了所有的自定义属性的值。
获取自定义属性的方法：node.dataset[info]或者node.dataset.info。
